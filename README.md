#  Sparta Education – 3 Week Data Engineering Bootcamp

##  Overview

This repository contains work completed during the **Sparta Education 3-week Data Engineering bootcamp**.  
The course provides an introduction to core data engineering concepts, tools, and workflows including:

- SQL (relational databases and querying)
- MongoDB (NoSQL databases)
- Databricks (data processing and analytics using Apache Spark)
- Data pipelines and ETL fundamentals

---

##  Objectives

The main goals of this bootcamp are to:

- Understand the difference between SQL and NoSQL databases
- Learn how to design and query relational databases using SQL
- Gain hands-on experience with MongoDB for document-based data storage
- Explore big data processing using Databricks and Apache Spark
- Build basic ETL (Extract, Transform, Load) pipelines
- Develop foundational data engineering skills for cloud environments

---

##  Topics Covered

###  SQL (Structured Query Language)

- Database design fundamentals
- SELECT, INSERT, UPDATE, DELETE
- JOINs and relationships
- Aggregations and filtering
- Normalisation basics

---

###  MongoDB (NoSQL Database)

- Document-based data structure (JSON/BSON)
- Collections and documents
- CRUD operations
- Querying nested data
- Differences between SQL and NoSQL systems

---

###  Databricks & Apache Spark

- Introduction to big data processing
- Distributed computing concepts
- Using notebooks (Python / SQL)
- Data transformation using Spark DataFrames
- Basic ETL workflows
- Overview of Delta Lake

---

###  Data Engineering Concepts

- ETL vs ELT pipelines
- Data lakes vs data warehouses
- Batch vs streaming data
- Introduction to cloud data platforms
