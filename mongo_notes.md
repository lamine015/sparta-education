# Introduction to MongoDB
``MongoDB`` is a NOSQL type database that stores data in JSON like ``documents`` called ``BSON`` (Binary JSON) which allows you to store data with flexible schema and provides querying and aggregation tools for access and analysis.

 unlike the other relational databases such as ``PostgreSQL``, mongoDB stores data in ``collections`` and ``documents`` rather than ``tables`` and ``rows``.

## Key difference
| Relational Database | MongoDB |
|---------------------|----------|
| Database | Database |
| Table | Collection |
| Row | Document |
| Column | Field |

## Example Document
``` 
{
    name: "Mohamed Aboudaoud",
    age: 30
    city: "London"
}
```
