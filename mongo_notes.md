# Introduction to MongoDB
``MongoDB`` is a NOSQL type database that stores data in JSON like ``documents`` called ``BSON`` (Binary JSON). unlike the other relationalt databases such as ``PostgreSQL``, mongoDB stores data in ``collections`` and ``documents`` rather than ``tables`` and ``rows``.

## Key difference
| Relational Database | MongoDB |
|---------------------|----------|
| Database | Database |
| Table | Collection |
| Row | Document |
| Column | Field |

## Example Document
``` 
{
    name: "Mohamed Aboudaoud",
    age: 30
    city: "London"
}
```
